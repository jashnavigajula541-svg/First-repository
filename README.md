# Hospital Management System in Java

This is a menu-driven Hospital Management System developed in Java.
It uses file handling to store patient records permanently.

## Features
✔ Add patient  
✔ View all patients  
✔ Search patient by ID  
✔ Delete patient  

## Technologies Used
- Java
- File handling
- OOP concepts
- CRUD operations

## How to Run
javac HospitalManagement.java
java HospitalManagement
